图神经网络定义 定义

图神经网络在推荐系统中的应用研究 | 第4页

图神经网络（Graph Neural Networks, GNNs）是一类专门用于处理图结构数据的深度学习模型。在推荐系统中，用户-物品交互自然形成二分图结构，因此GNN成为处理推荐问题的有力工具。

GNN核心公式 公式

图神经网络在推荐系统中的应用研究 | 第7页

h_v^{(l+1)} = σ(∑_{u ∈ N(v)} (1/c_{v,u}) · W^{(l)} · h_u^{(l)})

其中，h_v^{(l)}表示节点v在第l层的特征表示，N(v)表示节点v的邻居集合，c_{v,u}是归一化常数，W^{(l)}是可学习的权重矩阵，σ是非线性激活函数。

GNN核心操作代码 代码

图神经网络在推荐系统中的应用研究 | 第11页

def forward(self, x, edge_index):
    # 线性变换
    x = self.linear(x)
    
    # 消息传递
    x = self.propagate(edge_index, x=x)
    
    # 非线性激活
    x = self.act(x)
    
    return x

GNN层的前向传播函数实现，包含线性变换、消息传递和非线性激活三个核心步骤。这是GNN实现信息传递和聚合的关键部分。
